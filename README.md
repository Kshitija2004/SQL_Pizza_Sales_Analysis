# SQL_Pizza_Sales_Analysis

SQL project to analyze pizza sales data.

This project focuses on analyzing pizza sales data using SQL to extract meaningful business insights.
The goal is to understand sales performance, customer ordering behavior, revenue trends, and product popularity to help improve business decision-making in a pizza restaurant.

Objectives:

Analyze overall sales and revenue

Identify best-selling and least-selling pizzas

Understand customer order patterns

Analyze sales trends based on:
Pizza siz
Categore
Time (daily & monthly)
